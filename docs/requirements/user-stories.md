# User Stories

## Client

As a client,
I want to create a support ticket
so that I can report an issue.


As a client,
I want to view my ticket history
so that I can track previous issues.


## Support Agent

As a support agent,
I want to update ticket status
so that clients know the progress.


## Administrator

As an administrator,
I want to manage users
so that only authorized people can access the system.
